How to register a custom auto configuration
- mention the fully qualified name of the class under resources/META-INF/spring.factories file.
- @EnableAutoConfiguration will pickup the class from the above file
- example in file
org.springframework.boot.autoconfigure.EnableAutoConfiguration=com.baeldung.autoconfigure.CustomAutoConfiguration
